## 🤔 ¿Por qué OpenCode?

OpenCode es una plataforma emergente que permite a los desarrolladores crear, gestionar y compartir "skills" de inteligencia artificial de manera sencilla y eficiente. La necesidad de integrar habilidades de IA en nuestras aplicaciones ha crecido exponencialmente, y OpenCode se presenta como una solución accesible tanto para principiantes como para expertos.

## 🚀 Instalación de OpenCode CLI

Antes de poder crear skills, necesitamos instalar la herramienta de línea de comandos (CLI) de OpenCode. La CLI nos permite interactuar con la plataforma de manera fácil y rápida.

### Requisitos previos

- Node.js (versión 14 o superior)
- npm (gestor de paquetes de Node.js)

### Instalación paso a paso

Para instalar la CLI de OpenCode, sigue estos pasos:

1. **Abrir Terminal:** Inicia tu terminal de preferencia en tu sistema operativo.
2. **Ejecutar el comando de instalación:**

   ```bash
   npm install -g opencode-cli
   ```

   Este comando instalará la CLI de OpenCode de manera global en tu sistema, de modo que puedas acceder a ella desde cualquier directorio.

3. **Verificar la instalación:**

   Para asegurarte de que la instalación se realizó correctamente, ejecuta:

   ```bash
   opencode --version
   ```

   Deberías ver la versión instalada de OpenCode CLI.

## 💡 Creación de Skills en OpenCode

Una vez que hayas instalado la CLI, es hora de crear nuestro primer skill en OpenCode. Un "skill" es una funcionalidad o tarea específica que tu aplicación puede realizar usando inteligencia artificial.

### Configuración inicial

1. **Iniciar sesión en OpenCode:**

   Antes de crear un skill, necesitas autenticarte. Ejecuta el siguiente comando y sigue las instrucciones para iniciar sesión:

   ```bash
   opencode login
   ```

2. **Crear un nuevo proyecto de skill:**

   Ahora, vamos a crear un nuevo proyecto:

   ```bash
   opencode create skill mi-primer-skill
   ```

   Este comando creará una nueva carpeta llamada `mi-primer-skill` con todos los archivos necesarios para comenzar.

### Estructura del proyecto

Al crear un nuevo skill, OpenCode genera una estructura de carpetas básica:

- **`src/`**: Aquí se encuentra el código fuente de tu skill.
- **`tests/`**: Carpeta destinada a las pruebas unitarias.
- **`opencode.json`**: Archivo de configuración del skill.

### Implementación de un Skill básico

Vamos a implementar un simple skill que realiza una tarea de ejemplo: sumar dos números.

1. **Editar el archivo principal:**

   Navega al directorio `src/` y abre el archivo principal, normalmente `index.js`.

   ```javascript
   // src/index.js

   module.exports = {
   	suma: (a, b) => {
   		return a + b // Retorna la suma de dos números
   	},
   }
   ```

2. **Configurar el Skill:**

   Abre `opencode.json` y ajusta la configuración básica:

   ```json
   {
   	"name": "mi-primer-skill",
   	"version": "1.0.0",
   	"description": "Un skill simple que suma dos números",
   	"main": "src/index.js",
   	"scripts": {
   		"test": "npm test"
   	}
   }
   ```

### Pruebas y despliegue

Para asegurarnos de que nuestro skill funciona correctamente, es importante escribir pruebas.

#### Crear una prueba unitaria

1. **Escribir pruebas:**

   En la carpeta `tests/`, crea un archivo `index.test.js` y escribe lo siguiente:

   ```javascript
   // tests/index.test.js

   const { suma } = require('../src/index')

   test('suma de 2 + 3 debe ser 5', () => {
   	expect(suma(2, 3)).toBe(5)
   })
   ```

2. **Ejecutar pruebas:**

   Ejecuta las pruebas con el siguiente comando:

   ```bash
   npm test
   ```

   Si todo está correcto, deberías ver un mensaje indicando que las pruebas pasaron exitosamente.

#### Desplegar el skill

Finalmente, para hacer disponible tu skill en OpenCode:

```bash
opencode deploy
```

Este comando subirá tu skill a la plataforma de OpenCode, donde otros desarrolladores podrán utilizarlo.

## ⚡ Conclusión

Crear y desplegar skills en OpenCode es un proceso sencillo y directo que puede potenciar significativamente tus proyectos de IA.
